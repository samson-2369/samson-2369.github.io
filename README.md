# Cornell Security Jekyll Starter

This is a starter GitHub Pages site for Cornell Security using Jekyll-compatible files.

## Quick upload steps

1. Download and unzip this starter package.
2. Upload the contents into your `samson-2369.github.io` repository.
3. Commit the files.
4. In GitHub, go to **Settings > Pages**.
5. Set Source to **Deploy from a branch**.
6. Set Branch to **main** and folder to **/(root)**.
7. Save.

GitHub Pages will build the site automatically.
